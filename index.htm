PHASE 1: REVISED PROBLEM STATEMENT AND DATASET

Project: Canadian Mortgage and Housing Corporation (CMHC) Web Application

Ideal: An interactive web app that will show housing prices over time in a chosen area.

Reality : CMHC mostly uses paper maps and is looking to transfer to online maps in order to display more data that can still be easily understood. Paper maps are limited to its size, limiting small-scale maps in what information they can display.

Consequences: Without this web app CHMC will continue to have a difficult time displaying and sharing their data. This will hinder their abilities to use the data as a source in advocating for policy reform and presenting their research.

Proposal: Create an interactive web app that will be able to show how housing prices have changed over a period of time in a given area.

Dataset: For this web app, we searched for open source data in a .csv format that also included a Time field, as well as a date. The data file includes latitude and longitude. A dataset of number of housing starts in Canada, the provinces and all major metropolitan markets provided by CMHC was used. It was then modified to show a small portion of it by displaying the number of housing starts from 2012 to 2016 for five metropolitan cities.

PHASE 2: EXPLORING NETWORK ANALYSIS IN ARCGIS ONLINE AND ALTERNATIVE APPLICATION

ArcGIS Online a cloud-based mapping platform was examined in order to host the data layer. To enable the time, an ESRI product Time Aware, a Time-enabled confgurable app was utlized to create the primary web app. To enable the time feature, the csv data file was hosted as a layer and then referenced that layer into the map to enable the time feature.
